"use strict"; defines that js code should be executed in "strict mode"


Not Allowed in Strict Mode:
Using variable, without declaring it
Using an object, without declaring it
deleting a var / obj / function
duplicate parameter name
octal numeric literals
octal escape characters
writing to a readonly property
writing to a getonly property
deleting an undeletable property
eval,arguments cannot be used as variable
eval() is not Allowed to create var in the scope from which it was created
var can not be used before it is declared
eval() can not declare a var using the var or let keyword
